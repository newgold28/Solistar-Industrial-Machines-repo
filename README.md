## 🍳 About the Project

Solistar Industrial Machines supplies commercial kitchen equipment to businesses across Nigeria — from restaurants and hotels to bakeries and food vendors. This website was built to help them reach more customers online and make it easy for buyers to explore products and get in touch.

### What the site covers:
- Commercial kitchen equipment product catalog
- About section — company background and values
- Client testimonials
- Location/map for physical contact

## 🚀 Getting Started

### Prerequisites

Make sure you have these installed:
- [Node.js](https://nodejs.org/) (v18 or higher)
- npm or yarn

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/your-username/solistar-machines.git

# 2. Navigate into the project folder
cd solistar-machines

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```

Then open your browser and go to: `http://localhost:5173`

### Build for Production

```bash
npm run build
```

The output will be in the `/dist` folder, ready to deploy.

### Preview Production Build

```bash
npm run preview
```
